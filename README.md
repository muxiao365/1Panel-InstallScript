# 1Panel-InstallScript
## 简介
本项目旨在通过安装1Panel，配置Docker镜像加速（1Panel官方），并配置1Panel三方应用市场，适配Ubuntu或Debian及其对应发行版，为1Panel新用户提供更高效的服务器和部署环境。

## 功能

- 安装1Panel

- 配置Docker镜像加速

- 配置1Panel三方应用市场

- 三方应用市场自动更新

## 安装指南
- 操作系统：Ubuntu, Debian及其对应发行版
  - **仅适配Ubuntu或Debian及其对应发行版**
